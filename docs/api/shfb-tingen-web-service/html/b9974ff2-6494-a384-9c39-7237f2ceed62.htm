      Remarks

      This class doesn't do much actual work, and should remain fairly static. For the most part, it just hands information to Outpost31, where the heavy lifting is done.
      There are two methods in this class which are required and should not be modified:

      • GetVersion
      • RunScript(OptionObject2015, String)

      For more information about the Tingen Web Service, please see Tingen Web Service ProjectInfo.
