      Remarks

      The WriteStartLog() method is only for debugging purposes.

      If Avatar doesn't pass an origOptObj or origScriptParam, the Tingen Web Service will fail, and a critical log will be created.

      The AvtrSys setting specifies which Avatar System the Tingen Web Service will interface with.
      It is very important that AvtrSys is set correctly, otherwise lots of bad things can *and will* happen!

      The TngnWsvcMode must be set to enabled, disabled, or passthrough. Any other mode will behave as disabled.
      There is a special "AdminMode" that can be set in the Web.Config file, but this is only for administrative/development tasks. Additional information about this mode can be found in the Tingen Web Service Manual.

        The Tingen Web Service Manual: Modifying AvtrSys

      To modify the AvtrSys setting:
      1. Open the Web.Config file located in the root directory of the Tingen Web Service.
      2. Locate the AvtrSys setting within the applicationSettings section.
      3. Change the value of the AvtrSys setting to the desired value.
      4. Save the changes to the Web.Config file.
