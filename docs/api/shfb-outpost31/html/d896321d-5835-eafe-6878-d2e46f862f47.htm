      Remarks

        About this class

      The following types of logs are supported:
      • Critical(String, String)
      • Debug(String, String, String, String, Int32, String)
      • Debuggler(String, String)
      • Primeval(String, String)
      • Trace(Int32, String, String, String, String, Int32, String)

      Methods

      CriticalCritical logs indicate a significant failure with the Tingen Web Service.
      DebugDebug logs are used when developing the Tingen Web Service.
      DebugglerDebuggler logs are temporary logs used when you want to debug something specific.
      PrimevalPrimeval logs are used when the advanced logging infrastructure isn't available.
      TraceTrace Logs are used for debugging and tracking the flow of the application.
