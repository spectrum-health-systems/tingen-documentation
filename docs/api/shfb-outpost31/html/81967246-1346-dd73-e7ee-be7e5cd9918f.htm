      Remarks

      In order to successfully return an OptionObject to Avatar:
      1. The OptionObject ErrorCode must be set.
        The most common error codes are:
        • 1 - Used when you don't want the user to take an action, such as submitting a form.
        • 3 - Used when you would like to notify the user of something.
        • 4 - Used when you want warn the user they may want to make changes, and give them the option to stop further processing,or continuing.
      2. The OptionObject Error Message must be set.
        The message that is displayed to the user.
      3. All OptionObject components must be validated and formatted correctly.
        This is done via ToReturnOptionObject(Int32, String).
