      The default value is UAT.

      Remarks

      It is very important that AvtrSys is set correctly, otherwise lots of bad things can and will happen!

      Common Avatar Systems are: UAT, LIVE, SBOX, and BUILD. However, the Tingen Web Service is only
      able to interface with the UAT and LIVE Avatar Systems.

        How-To

      Modify the AvtrSys setting

      1. Open the Web.Config file located in the root directory of the Tingen Web Service.
      2. Locate the AvtrSys setting within the applicationSettings section.
      3. Change the value of the AvtrSys setting to the desired value.
      4. Save the changes to the Web.Config file.
      For example, to set the Avatar System to `LIVE`:
      C#
      <setting name="AvtrSys" serializeAs="String">
    <value>LIVE</value>
</setting>
      Changes to the AvtrSys setting will be applied the next time the Tingen Web Service is executed.
