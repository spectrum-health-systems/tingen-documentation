      Remarks

      Common Avatar Systems include SBOX, UAT, and LIVE.

        How-To Modify the AvatarSystem setting

      To modify the AvatarSystem that the Tingen Web Service will interface with:

      1. Open the Web.Config file located in the root directory of the Tingen Web Service.
      2. Locate the AvatarSystem setting within the applicationSettings section.
      3. Change the value of AvatarSystem setting to the desired value.
      4. Save the changes to the Web.Config file.
      For example, to set the Avatar System to `LIVE`:
      C#
      <setting name="AvatarSystem" serializeAs="String">
    <value>LIVE</value>
</setting>
      Changes to the AvatarSystem setting will be applied the next time the Tingen Web Service is executed.
