      Remarks

      The Script Parameter sent from Avatar contains all of the information that the Tingen Web Service needs to do work.

      The script parameter comprised of the following components:

      • The requested Module (the namespace)
      • The requested Command (the class)
      • The requested Request (the method)
      • The requested Option
      In addition:
      • The maximum length of a script parameter is 50 characters
      • Components are separated by a "."
      • Prototype parameters start with "_p"
      Examples:
      • Admin.Status.Current
      • FormAccess.Deny.SysCodeDoc
      • _pRequest.Under.Development


      Script parameters must one of the current valid parameters.

      For more information about Outpost31, please see ProjectInfo.
