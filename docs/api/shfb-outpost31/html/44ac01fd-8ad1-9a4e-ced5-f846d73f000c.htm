      Remarks

        Administrative mode options

      The following administrative modes are available:
      • Deploy
      • Refresh
      • Regression

        How-To

      Modify the AdminMode setting

      1. Open the Web.Config file located in the root directory of the Tingen Web Service.
      2. Locate the AvtrSys setting within the applicationSettings section.
      3. Change the value of the AdminMode setting to the desired value.
      4. Save the changes to the Web.Config file.
      For example, to set the AdminMode to `RegressionTest`:
      C#
      <setting name="AdminMode" serializeAs="String">
    <value>RegressionTest</value>
</setting>
      Changes to the AdminMode setting will be applied the next time the Tingen Web Service is executed.
